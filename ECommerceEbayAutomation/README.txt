Problem Title: E-Commerce Automation

Description:

Get 5 mobiles from a popular online store, which has:

1. Price less than Rs. 30000

2. Mobiles which are newly arrived.

( Suggested site : flipkart.com, however you are free to choose any other legitimate shopping site.)


Specifications:

1. Java version : jdk 8.0_111, build "1.8.0_111-b14"

2. Eclipse Version : Mars.2 Release (4.5.2), Build id: 20160218-0600

3. Selenium version : 3.141.59

4. TestNG version : 6.11

5. Google Chrome version : 86.0.4240.111

6. Opera version : 72.0.3815.148

7. Website used : https://www.in.ebay.com/


Solution:

1. Navigate to https://www.in.ebay.com/ website

2. Click the Main Category as "Electronics" 

3. Click the subcategory as "Cell Phones, Smart Watches & Accessories"

4. Click the subcategory as "Cell Phones & Smartphones"

5. Set Minimum price as "0" and Maximum price  as "404" in textbox and click on the ">" button

6. Select sort type as "Time: newly listed"

7. Take screenshot of the page

8. Store mobile details to excel file

9. Close the opened browser

################################################################################################################################################
INFO: The currency format of ebay.com is in U.S. Dollars. The maximum price is set as $404, which is approximately equal to 30000 Indian Rupees.
################################################################################################################################################
